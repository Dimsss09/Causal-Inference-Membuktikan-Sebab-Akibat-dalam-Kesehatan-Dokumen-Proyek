# Causal Inference — Membuktikan Sebab-Akibat dalam Kesehatan — Dokumen Proyek

<aside>
📌

Folder dokumen untuk proyek **Causal Inference** — menjawab pertanyaan *"apakah X benar-benar MENYEBABKAN Y?"* dari data kesehatan observasional (bukan sekadar korelasi/prediksi). Berisi semua dokumen untuk mengerjakan proyek dari nol sampai selesai. Tanpa hardware — cukup data publik.

</aside>

<aside>
🎯

**Target default proyek ini: "Apakah pemasangan Right Heart Catheterization (RHC) memengaruhi mortalitas pasien ICU?"** menggunakan dataset RHC (Connors et al., 1996) yang publik & merupakan studi kasus klasik causal inference medis. Dokumen tetap bisa diadaptasi ke pertanyaan lain (mis. efek merokok pada tekanan darah via NHANES) — lihat catatan di tiap dokumen.

</aside>
